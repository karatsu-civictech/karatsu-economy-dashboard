# 唐津市 経済動向ダッシュボード

唐津市が公開している月次経済動向データ（2010年〜）を、**BIツールのようにソート・フィルター・グラフ**で自由に閲覧できる可視化Webアプリです。シビックテックによる非公式プロジェクト。

元データはGoogleスプレッドシートの表形式（指標が縦、年が横、月ごとにタブ）で視認性が低いため、これを構造化データに変換し、誰でも直感的に動向を読み取れるようにしています。

## 特徴

- **フィルター**: 期間（年範囲）・カテゴリ・指標を自由に絞り込み
- **集計切替**: 単月（任意の月）/ 年平均 / 年合計
- **テーブル**: 列見出しクリックで並べ替え、最大・最小・直近増減を併記、**CSVダウンロード**
- **グラフ**: 年次/月次の折れ線、複数指標の重ね合わせ、単位が異なる指標を比べるための**指数化（基準=100）**、コロナ影響期（2020〜2021年）のハイライト
- 完全な静的サイト（サーバ不要）。Cloudflare Pages / GitHub Pages / Vercel などに無料でデプロイ可能

## データ

- 出典: 唐津市 経済動向データ（Googleスプレッドシート）
- カテゴリ: 観光 / 企業活動 / 一次産業 / 雇用 / 個人消費 / 広域交流
- 27指標 × 2010〜2026年 × 12ヶ月

## 開発

```bash
npm install

# データ生成（スプレッドシートから取得 → public/data.json）
npm run data            # ネットから最新を取得（raw-data/ にキャッシュ）
npm run data:offline    # raw-data/ のキャッシュのみ使用

# 開発サーバ
npm run dev

# 本番ビルド
npm run build           # dist/ に出力
npm run preview         # ビルド結果をローカル確認
```

## データ更新の流れ

元スプレッドシートが更新されたら以下を実行するだけで反映されます。

```bash
npm run data    # 12タブを再取得して public/data.json を再生成
npm run build   # 再ビルド
```

`scripts/build-data.mjs` がCSVを取得・整形します。スプレッドシートのID・タブ(gid)対応もこのファイル内で定義しています。

## 構成

```
scripts/build-data.mjs   データ取得・整形パイプライン（CSV → data.json）
raw-data/*.csv           取得したCSVのキャッシュ（12ヶ月分）
public/data.json         アプリが読み込む整形済みデータ
src/
  App.tsx                画面全体・状態管理
  components/
    Sidebar.tsx          フィルター（期間・集計・カテゴリ・指標）
    DataTable.tsx        ソート可能なテーブル＋CSV出力
    TrendChart.tsx       折れ線グラフ（年次/月次・指数化）
  lib/data.ts            データ読み込み・集計・整形ヘルパ
```

## 注意

- 元データには欠測（施設改修・集計方法変更など）があります。各指標の「※」マークにカーソルを合わせると注釈が表示されます。
- 年平均・年合計は、欠測月を除いて算出しています。
- 本アプリは市公式のものではありません。数値の正確性は元データをご確認ください。
